# kino_store_posV2
Punto de venta web para tienda de gorras y playeras

## Generar ejecutable para Linux Mint

Desde `kino_pos/`:

```bash
npm install
npm run dist:linux
```

Los instalables se generan en `kino_pos/dist/`:
- `*.deb` para instalar en Linux Mint.
- `*.AppImage` para ejecutar sin instalar.
